# flowstate

## What is it?

- _**flowstate**_ is inpired by Flowmodoro (or Flowtime Technique), which is a flexible productivity method designed to maintain focus without the rigid, forced breaks of the Pomodoro Technique. 
- Instead of counting down, it uses a timer that counts up, allowing you to work until you naturally lose focus, then taking a break proportional to the work duration.
- You also have _**lo-fi radios**_ to play in the background while you do your work.

## Tools used:
